# doanteam

Đây là dự án trang web giới thiệu bản thân đơn giản.

## Cách sử dụng

1. Mở file `index.html` trong trình duyệt web để xem trang web.
2. Để chạy server local (khuyến nghị), sử dụng một server HTTP đơn giản:
   - Nếu bạn có Python: `python -m http.server 8000`
   - Nếu bạn có Node.js: cài đặt `http-server` và chạy `http-server`
3. Truy cập `http://localhost:8000` trong trình duyệt.

## Tùy chỉnh

- Chỉnh sửa `index.html` để thay đổi nội dung giới thiệu.
- Sửa `style.css` để thay đổi giao diện.
- Thêm chức năng trong `script.js` nếu cần.

## Khắc phục sự cố

- Đảm bảo tất cả file (index.html, style.css, script.js) ở cùng thư mục.
- Nếu không hiển thị đúng, kiểm tra đường dẫn trong HTML.
